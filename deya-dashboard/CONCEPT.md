# Deya Dashboard
# Веб-интерфейс для управления OpenClaw инстансом

## Концепция

После установки OpenClaw пользователь открывает браузер → http://localhost:8000 → видит дашборд Деи.

## Страницы

### 1. Главная (Chat)
- Чат-интерфейс с Деей
- История сообщений
- Быстрые действия (кнопки)
- Статус инстанса (online/offline)

### 2. Дашборд (Dashboard)
- Статистика: сообщения, задачи, память
- График активности
- Последние действия
- System health (CPU, RAM)

### 3. Скиллы (Skills)
- Список установленных скиллов
- Кнопка "Установить" (загрузка .skill файла)
- Кнопка "Удалить"
- Описание каждого скилла
- Статус: активен/ошибка

### 4. Память (Memory)
- Просмотр MEMORY.md
- Дневники по датам
- Поиск по памяти
- Редактирование

### 5. Каналы (Channels)
- Telegram подключение
- Список постов
- Кнопка "Написать пост"
- Просмотр статистики канала

### 6. Задачи (Tasks)
- Cron-джобы (список)
- Добавить новую задачу
- Логи выполнения
- Background processes

### 7. Настройки (Settings)
- Конфигурация модели (Kimi/OpenRouter)
- API ключи
- Часовой пояс
- Пароль для доступа к дашборду
- Backup/Restore

## Технологии

- **Backend:** FastAPI (уже есть)
- **Frontend:** Jinja2 шаблоны + HTMX (для динамики без React)
- **Стили:** Tailwind CSS (уже используется)
- **JS:** Alpine.js (лёгкий, быстрый)

## Структура

```
deya-dashboard/
├── main.py              # FastAPI приложение
├── static/
│   ├── css/
│   │   └── style.css    # Tailwind
│   └── js/
│       └── app.js       # Alpine.js
├── templates/
│   ├── base.html        # Базовый шаблон
│   ├── index.html       # Главная (чат)
│   ├── dashboard.html   # Дашборд
│   ├── skills.html      # Скиллы
│   ├── memory.html      # Память
│   ├── channels.html    # Каналы
│   ├── tasks.html       # Задачи
│   └── settings.html    # Настройки
└── requirements.txt
```

## Особенности

1. **Авторизация** - простая basic auth или JWT
2. **Real-time** - WebSocket для чата
3. **Mobile-first** - адаптивный дизайн
4. **Тёмная тема** - как у AI Router
5. **Deya branding** - логотип, цвета, шрифты

## Быстрые действия на главной

- "Сгенерировать пост"
- "Проверить почту"
- "Сделать backup"
- "Перезапустить"

## Установка

```bash
pip install -r requirements.txt
python main.py
# Открыть http://localhost:8000
```

## Интеграция с OpenClaw

Дашборд будет:
- Читать память из файлов
- Запускать скрипты из skills/
- Использовать OpenClaw CLI через subprocess
- Показывать логи из stdout
