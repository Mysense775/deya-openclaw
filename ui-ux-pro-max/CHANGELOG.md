# Changelog UI-UX-Pro-Max

## [0.3.0] - 2025-02-26

### Added
- `Card.tsx` — полноценный компонент карточки (6 вариантов)
- `Input.tsx` — поле ввода с иконками, error/success states, password toggle
- `Badge.tsx` — бейджи с градиентом, glow, dot indicator, removable
- `canvas-preview.html` — визуальный превью всех компонентов
- Обновлён SKILL.md с workflow и примерами
- Обновлён README.md с документацией по всем компонентам

### Components Ready
- ✅ Button — все варианты, иконки, loading
- ✅ Card — default, gradient, glass, dark, elevated, outline
- ✅ Input — default, filled, outline, underline, error, success
- ✅ Badge — default, primary, success, warning, danger, gradient, glow

## [0.2.0] - 2025-02-25

### Added
- `code-generator.py` — генератор React-компонентов из описания
- Поддержка 4 типов: button, card, input, badge
- Распознавание русского и английского языка
- Автоопределение вариантов, размеров, форм
- `deya-touch.md` — гайд по добавлению вайба Деи в дизайн
- `Button.tsx` — пример готового компонента
- README с примерами использования

### Features
- Генерация кода по текстовому описанию
- Tailwind CSS классы
- TypeScript-ready компоненты
- Дизайн-токены AI Router
